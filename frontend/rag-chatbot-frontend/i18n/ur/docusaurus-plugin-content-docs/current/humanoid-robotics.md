# انسان نما روبوٹکس

## انسان نما روبوٹکس کیا ہے؟

انسان نما روبوٹ ایسے روبوٹس ہیں جن کی تعمیر انسانوں کی طرح حرکت کرنے اور بات چیت کرنے کے قابل ہونے کے لیے کی گئی ہے۔ یہ روبوٹکس کے ایک اہم رجحان کی نمائندگی کرتے ہیں۔

### اہم خصوصیات

- **(human-like form)**: انسان کی طرح شکل
- **(bipedal locomotion)**: دو پاؤں پر چلنے کی صلاحیت
- **(manipulation)**: اشیاء کو اٹھانے اور ہیرا پھیری کرنے کی صلاحیت
- **(interaction)**: انسانوں کے ساتھ بات چیت کرنے کی صلاحیت

### اہم ترقیات

- **1990s-2000s**: اعلی کنٹرول سسٹم

انسان نما روبوٹکس میں کئی چیلنج ہیں:

- **توازن**: دو پاؤں پر چلتے وقت توازن برقرار رکھنا
