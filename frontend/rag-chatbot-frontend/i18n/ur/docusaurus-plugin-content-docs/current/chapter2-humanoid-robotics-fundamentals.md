# انسان نما روبوٹکس کے فنڈامینٹلز

## تعارف

انسان نما روبوٹکس وہ شعبہ ہے جہاں روبوٹس کو انسانوں کی طرح حرکت کرنے اور کام کرنے کے قابل بنایا جاتا ہے۔ یہ روبوٹکس کا ایک اہم رجحان ہے۔

## اہم خصوصیات

### انسان کی طرح شکل
- دو ہاتھ
- دو پاؤں
- ایک سر
- ایک躯 trunk

### حرکت
- دو پاؤں پر چلنا
- توازن برقرار رکھنا
- ڈھلوان پر چلنا

### ہیرا پھیری
- اشیاء کو اٹھانا
- چھوٹی چیزوں کو سنبھالنا
- پیچیدہ حرکات

## چیلنج

### تکنیکی چیلنج
- توازن
- کنٹرول
- بجلی کی کھپت
- حفاظت

### مستقبل کی ترقیات
- بہتر کنٹرول الگورتھم
- جدید سینسرز
- اعلی مصنوعی ذہانت

## اطلاقات

### موجودہ اطلاقات
- سروس روبوٹ
- تعلیمی روبوٹ
- تفریحی روبوٹ

### مستقبل کی امکانی اطلاقات
- گھریلو مدد
- صحت کی دیکھ بھال
- صنعتی استعمال
